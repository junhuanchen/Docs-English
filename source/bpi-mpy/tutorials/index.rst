基础教程
========================================

MicroPython是 Python 3 语言的精简实现 ，包括Python标准库的一小部分，经过优化可在微控制器和受限环境中运行。

.. Hint::

    如果你从没学过 Python3，建议你先从这里 `Python 3 菜鸟教程 <https://www.runoob.com/python3/python3-tutorial.html>`_ 进行学习 Python3 的软件基础知识，再过度到硬件部分会比较好。
